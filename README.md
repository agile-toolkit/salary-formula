# Salary Formula

A transparent salary calculation tool based on Management 3.0's Salary Formula approach — formula-driven compensation with full visibility into every factor.

Part of the [Agile Tools](https://github.com/bthos) suite built on Management 3.0 and ICAgile source materials.

## Stack
React 18 · TypeScript · Vite · Tailwind CSS · react-i18next (EN/RU)

## Development
```bash
npm install
npm run dev
```

## Deploy
GitHub Pages via GitHub Actions on push to `main`.
